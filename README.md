# Engineering Data Analyzer

## Overview
**Engineering Data Analyzer** is a Python-based project focused on analyzing real-world engineering datasets. The core goal is to automate the loading, cleaning, analysis, visualization, and reporting of data, helping engineers extract valuable insights from complex technical data.

This project is designed as a high-standard learning experience, strictly following professional software development practices to create a modular, clean, and scalable application.

## Key Features
* 🔄 **Automated Data Pipeline:** End-to-end data loading and cleaning.
* 📊 **Advanced Analytics:** Statistical analysis tailored for technical engineering data.
* 📈 **Clear Visualization:** Automated generation of charts and reports.
* 🛠️ **Professional Architecture:** Built with modularity, virtual environments, and clean code principles.

## Dataset Information
* **Source:** Red Eléctrica de España (REE) - ESIOS Public Portal
* **Dataset Name:** Generación y Consumo
* **Content:** Real-time electricity demand ($MW$) for the Spanish Peninsula grid.
* **Data Format:** CSV (Semi-colon separated `;`)
* **Time Resolution:** 5-minute intervals
* **Download Date:** 04/07/2026 (20:14 CEST)
* **Date Range:** From 03/07/2026 (21:00 CEST) until 04/07/2026 (19:55 CEST)
* **License:** Open Data / Public Domain (Terms of use by REE/ESIOS)
